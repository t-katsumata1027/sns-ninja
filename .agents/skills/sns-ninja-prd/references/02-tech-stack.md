# 2. 2026年最適技術スタック選定

## 2.1 選定基準（ゼロベース）

| 要件 | 選定理由 |
|------|----------|
| 大規模アカウント管理 | 数千アカウントの同時運用 |
| 高速画像・動画生成 | API応答時間 < 3秒 |
| 検出回避 | 人間行動シミュレーション |

| マルチテナント | 買い切り型商品として再販可能 |

## 2.4 技術選定の根拠

### Next.js 14+ (App Router)
- **Server Components**で初期読み込みを高速化
- **Edge Functions**でグローバル分散
- **API Routes**でバックエンドを統合

### Supabase + PostgreSQL
- **Row-Level Security**でマルチテナント分離をネイティブサポート
- **Realtime**機能でダッシュボード更新
- **Auth**機能で認証を簡略化

### Playwright + Stealth
- **検出回避**: WebDriver検出を回避
- **人間行動**: ランダムスクロール、クリック間隔
- **指紋偽装**: User-Agent、Viewport、Timezone

### Gemini 2.5 Pro
- **100万トークン**コンテキストウィンドウ
- **マルチモーダル**: テキスト・画像・動画
- **Google AI Studio**との統合
