# 5. アンチBAN・セキュリティ要件

## 5.1 人間行動シミュレーション
- **アクション間隔**: 投稿やリプライの間にランダムな待ち時間を設定。[cite: 90, 92, 94, 614]
- **レート制限**: BullMQ + Redis で管理（X: 30-50いいね/時、IG: 200DM/時を厳守）。[cite: 46, 775, 796]

## 5.2 指紋偽装
- **Playwright Stealth**: ブラウザ自動化の検知を回避。[cite: 65, 166, 217, 683]
- **住宅用プロキシ**: IPBurger 等の住宅用プロキシプールを使用し、IPを分散。[cite: 66, 171, 322]
