# 1. プロジェクト概要・収益化戦略

## 1.1 ビジョン

**SNS Ninja**は、Human-in-the-Loopによる完全自動化インフラを提供し、個人開発者向けに買い切り型商品として販売するSNSアフィリエイト自動化システムです。

### 収益目標
| フェーズ | 期間 | 目標収益 |
|---------|------|---------|
| Phase 1 | 30日 | $500-1,000/月 |
| Phase 2 | 3ヶ月 | $3,000-5,000/月 |
| Phase 3 | 6ヶ月 | $10,000+/月 |

## 1.2 X（旧Twitter）戦略

### インテリジェント・リプライ・エンジン
- **Gemini 2.5 Pro**で文脈認識型返信を生成
- 1日最大300件のリプライ対応
- 自動化と手動のハイブリッド運用（重要な関係は手動維持）

### 投稿頻度・タイミング（アンチBAN対策）
| アクション | 制限 | 間隔 |
|-----------|------|------|
| いいね | 30-50/時 | 90-180秒ランダム |
| フォロー | 5-15/時 | 30分-3時間ランダム |
| リプライ | 3-8/時 | 3-5分ランダム |
| リツイート | 5-10/時 | 2-4分ランダム |

### 収益化パス
1. **デジタル商品・アフィリエイトリンク**: $500-5,000/月
2. **X広告分配プログラム**: 収益共有
3. **スポンサードコンテンツ**: $100-500/投稿

## 1.3 Instagram戦略

### コンテンツレベル
| 種別 | 優先度 | 特徴 |
|------|--------|------|
| フィード投稿 | 高 | AI生成テキスト＋画像 |
| リール | 高 | AI動画（Runway/HeyGen） |
| ストーリーズ | 中 | テンプレート活用 |
| DM自動応答 | 高 | コンバージョン向上 |

### AI生成コンテンツ
- **画像**: Flux 2 / Stable Diffusion 3.5 / Gemini 2.5 Pro Image
- **動画**: Runway Gen-3 / HeyGen（フェイスレス動画）
- **音声**: ElevenLabs

### DM制限（2024年制変更）
- **1時間あたり200通**（以前の5,000通から削減）
- スマートキューで超過分を翌時間に送信
- キーワードトリガーで優先度付与：
  - **購入・価格関連** > **詳細・問い合わせ** > **一般的な質問**

## 1.4 クロスプラットフォームシナリオ

1. **Xバイラル → Instagram**: プロフィール誘導でフォロワー獲得
2. **Instagram Reels → X**: 長文解説へ誘導
3. **統合DB**: リード統合・重複排除

## 1.5 マルチテナント構成

- **小規模テナント**: 共有データベース（プールモデル）
- **大規模テナント**: 専用データベース（シロモデル）
- **分離方法**: PostgreSQL Row-Level Security (RLS)
